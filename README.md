# Smart-Fitness-Tracker-Barbell-Exercise-Classifier
Ongoing project

#Barbell Exercise Classification ML

A machine learning project that classifies different barbell exercises using accelerometer and gyroscope sensor data collected from wearable devices.

#Project Overview
This project follows the video series by Dave Ebbelaar and aims to build a fitness tracker capable of recognizing and classifying various barbell exercises based on motion sensor data.
